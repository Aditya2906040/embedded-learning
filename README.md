# Embedded Learning

A curated collection of my handwritten notes and reference material documenting my journey through Embedded Systems, Electronics, IoT, Networking, and Linux.

The goal of this repository is to build a structured knowledge base covering both theoretical concepts and practical engineering topics, with an emphasis on understanding how hardware and software interact in real embedded systems.

## Topics Covered

* Electronics
* Digital Electronics
* Computer Architecture
* Embedded C/C++
* Microcontrollers
* Sensors & Actuators
* Communication Protocols

  * UART
  * SPI
  * I²C
  * CAN
  * USB
* Wireless Communication
* Real-Time Operating Systems (RTOS)
* Embedded Linux
* Device Drivers
* Networking
* Security
* Debugging & Testing
* PCB Design
* Datasheets & Reference Manuals
* Schematics & Hardware Design

## About the Notes

The notes in this repository are handwritten and organized by topic. They are intended to serve as concise references and learning material rather than complete textbooks.

Where appropriate, topics are supplemented with:

* Block diagrams
* Timing diagrams
* Communication waveforms
* Architecture diagrams
* Datasheet references
* Practical engineering notes

## Disclaimer

These notes are created as part of my personal learning journey. They may evolve over time as my understanding deepens and I revisit topics.
